Locked decisions to honor in this phase:
- D-01: Preserve the canonical `manifest` / `graph` / `index` bundle architecture and keep the public contract additive at `0.2.0`.
- D-02: Deterministic parsing remains the trusted baseline; Phase 2 must not introduce agent-inferred semantics into the canonical artifact.
- D-03: Treat `external_dependency` as the Phase 2 citation object model; do not add a separate `citation` node kind unless implementation proves it is impossible.
- D-04: Source anchors must be the serialized `filePath`, `startLine`, and `endLine` fields already present on `MathNode`, not a new sidecar anchor system.
- D-05: Separate edge provenance from evidence so deterministic relations can carry both machine-filterable provenance and user-facing explanatory evidence.
- D-06: Keep Phase 2 scoped to canonical artifact quality and compatibility; no Phase 3 explorer workflow redesign and no Phase 4 enrichment storage.
